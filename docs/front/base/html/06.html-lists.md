# HTML 列表
代码示例
``` html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>html列表</title>
	</head>
	<body>
		无序列表:
		<ul>
		  <li>无序1</li>
		  <li>无序2</li>
		  <li>无序3</li>
		</ul>
		有序列表:
		<ol>
		  <li>有序1</li>
		  <li>有序2</li>
		  <li>有序3</li>
		</ol>
		自定义列表：
		<dl>
			<dt>项目1</dt>
			<dd>描述1</dd>
			<dt>项目2</dt>
			<dd>描述2</dd>
		</dl>
	</body>
</html>
```
::: tip 实例解析
* &lt;ul&gt; 标签定义无序列表 ，&lt;li&gt;定义列表项的内容
* &lt;ol&gt; 标签定义有序列表 ，&lt;li&gt;定义列表项的内容
* &lt;dl&gt; 标签定义自定义列表 ，&lt;dt&gt;定义项目，&lt;dd&gt;定义描述
:::
